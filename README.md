##Proceso

- si no es negativo y uno de los números ingresados es negativo
    -(a < 0 && b > 0) ó (a > 0 && b < 0)
        - muestra valor 1

- si es negativo y los numeros ingresados son negativos
    -(a < 0) y (b < 0)
        - muestra valor 1

- sino es asi
    - muestra valor 0
